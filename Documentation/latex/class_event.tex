\doxysubsection{Detailed Description}
An event is an object which stores functions which are called when a class calls a raise. For memory limit reasons, only 5 listeners are subscribable, though this limit can be changed by modifying the hard-\/coded constant "{}\+MAXIMUM\+\_\+\+LISTENERS"{} Listener functions should always be void functions only accepting a void\texorpdfstring{$\ast$}{*} argument
