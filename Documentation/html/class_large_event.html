Detailed Description

Event which supports more listeners. By default, the value of maximum listeners is set 10 times higher than a normal event.

An event is an object which stores functions which are called when a class calls a raise. For memory limit reasons, only 5 listeners are subscribable, though this limit can be changed by modifying the hard-coded constant "MAXIMUM_LISTENERS" Listener functions should always be void functions only accepting a void* argument
